The project aims to create a dynamic content website where users can access various types of content 
such as news, product launches, special offers, and more. The website will have a clean and organized 
layout with a header, navigation bar, main content area, and footer. Content will be dynamically injected 
into the webpage using JavaScript, and the layout will be styled using CSS. 

Feature List (User Requirements): 

1. Display Latest News: 
	- Show the latest news updates on the homepage. 
	- Each news item should include a title, content, category, date, author, and rating. 
	- Allow users to view more details by clicking on the news item. 

2. Showcase Product Launches: 
	- Display new product launches on a dedicated page. 
	- Each product launch should include a title, content, category, date, author, and rating. 
	- Provide a link to learn more about each product. 

3. Highlight Special Offers: 
	- Feature special offers prominently on the website. 
	- Each special offer should include a title, content, category, date, author, and rating. 
	- Allow users to access more information about the special offers. 

Software Requirements (Technical Tasks):

1. HTML Structure: 
	- Create HTML structure with header, navigation, main content area, and footer. 
	- Use semantic HTML elements for better accessibility and SEO. 
	- Include appropriate IDs and classes for styling and scripting purposes. 

2. CSS Styling: 
	- Style the header, navigation bar, main content sections, and footer. 
	- Apply responsive design principles to ensure compatibility with different devices and screen sizes. 
	- Use external CSS file to separate styles from HTML. 

3. JavaScript Functionality: 
	- Define a dynamicData array to store content data. 
	- Write a JavaScript function to inject content into the HTML based on the dynamicData array. 
	- Implement event handling to trigger content injection when the page loads. 

4. Content Display: 
	- Format the content layout to include titles, content, categories, dates, authors, and ratings. 
	- Dynamically generate HTML elements for each content item using JavaScript. 
	- Ensure proper alignment and spacing for readability. 

5. Linking: 
	- Create internal links within the website to navigate between different sections. 
	- Link to external resources for more information on news, product launches, and special offers. 
	- Use anchor tags with appropriate href attributes. 

By following these requirements, the project will deliver a functional and user-friendly dynamic content 
website that meets the needs of users seeking the latest news, product launches, and special offers.
